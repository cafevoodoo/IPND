Lesson 1 Notes

HTML

HTML is the glue that holds everything together. It is made up of the following components:

  • 1. Text content (what you see)
  • 2. Markup (what it looks like)
  • 3. References to other documents (images, videos)
  • 4. Links to other pages

Tags

HTML markup contains tags. There are opening and closing tags which are virtually identical minus the forward slash at the beginning of the closing tag. A complete tag is known as an element.
Italics was referred to as the 'em' tag. I learned it as i but both work.

Attributes

  • Tags can have attributes
  • Attributes will have values
  • Example is the anchor tag and HREF attribute

Images

Images are handled using the (img) tag. The img attribute is src which is important to note the since it is still calling at a specific URL, but is different from href attribute the anchor tag uses. The img tag has another attribute alled alt (alternative) that is important for the blind, but I learned it as being useful for SEO. Image tags don't have content so they do not have a closing tag, also known as a void tag.

Whitespace

  • HTML isn't like a word processor (WYSIWYG), it doesn't understand returns and tabs
  • br is a void tag allows for line breaks
  • p tag (paragraph) which is a block element. Renders group of text as paragraph

Inline versus Block Elements

Inline elements do not start on a new line, whereas block elements start on a new line and force anything after on a new line. The br tag is inline element that doesn't necessarily follow that rule. It forces whatever comes next on a new line. Block elements are like invisible boxes that can have height and width.

Boxes and Containers

Container elements only group things they contain together. Examples are:
  • span and div
  • span is inline
  • div is block

Structure of HTML Document

  • DOCTYPE - Defines the type of document (used to be complex).
  • HTML - Encompasses the entire document.
  • HEAD - Includes Java, CSS, and metadata along with the TITLE.
  • TITLE - Title of the page. Is displayed in browser window.
  • BODY - Contains contents of the document. Area that this course will focus.
